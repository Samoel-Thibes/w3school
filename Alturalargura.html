CSS Configuração de altura e largura

As propriedades height e width são usadas para definir e a largura de um elemento

As propriedades de altura e largura não incluem preenchimento, borda ou margens. Ele define a altura/largura da área dentro do preecnhimento, borda e margem do elemento.


Valores de altura e largura CSS

As propriedades height e width podem ter os seguintes valores:

  • auto - Este é o padrão. O navegador calcula a altura e a largura
  • lenght - Define a altura/largura em px, cm, etc.
  • % - Define a altura/largura em porcentagem do bloco que contém
  • intial - Define a altura/largura para seu valor padrão
  • inherit - Altura/largura será herdada de seu valor pai

Configurando largura máxima

A max-width propriedade é usada para definir a largura máxima de um elemento.

O max-width pode ser especificada em valores de comprimento, como px, cm, etc., ou em porcentagem (%) de bloco qie o contém, ou definido como nenhum (este é o padão. Significa que não há largura máxima).

O problema acima div ocorre quando a janela do navegador é menor que a alrgura do elemento (500px).O navegador então adiciona uma barra de rolagem horizontal à página.

Usar max-width, em vez disso, nesta situação, melhorará o manuseio de janelas pequenas pelo navegador.

Dica: Arraste a janela do navegador para uma largura menor que 500px para ver a diferença entre as duas divs!
