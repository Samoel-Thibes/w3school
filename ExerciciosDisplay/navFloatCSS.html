Itens de lista flutuante

Outra maneira de criar uma barra de navegção horizontal é flutuar os elementos li e especificar uma layout para os links

  • Home
  • News
  • Contact
  • About

Nota:Se !DOCTYPE não for especificado, itens flutuantes poderão produzir resultados inesperados.

Uma cor de fundo é adicionada aos links para mostrar a área do link, Toda a área do link é clicavel, não apenas o texto.

Nota: overflow:hidden é adicionando ao elemento ul para evitar que os elementos li saiam da lista.
