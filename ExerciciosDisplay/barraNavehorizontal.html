Barra de navegação horizontal

Existem duas maneiras de criar uma barra de navegaçao horizintal. usando itens de lista embutidos ou flutuantes.

itens da lista embutida

Uma maneira de construit uma barra de navegação horizontal é especificar os elementos li como embutidos. além di código "padrão".

  • Home
  • News
  • Contact
  • About

Exemplo explicado:

display: inline; - Por padrão, os elementos li são elementos de bloco. Aqui, removemos as quebras de linha antes e depois da cada item da lista, para exibi-lis em uma linha
