Bordas CSS

Estilos de Bordas CSS

A border-style propriedade especifica que tipo de borda exibir

  • dotted- Defiine uma borda pontilhada
  • dashed- Define uma borda tracejada
  • solid- Define uma borda sólida
  • double- Define uma borda dubla
  • grove- Define uma borda ranhurada 3D
  • ridge- Define uma borda estriada 3D
  • inset- Define uma borda inserida 3D
  • outset- Define uma borda inicial 3D
  • none- Não define fronteira
  • hidden- Define uma borda oculta
