Preenchimento CSS

O preenchimento é usado para criar espaço ao redor do conteúdo de uma elemento, dentro de quaisquer borda definida

Este elemento possui preenchimento de 70px.

Preenchimento - Lados Individuais

CSS possio propriedades para especificar preenchimneto de cada lado de uma elemento:

  • padding-top
  • padding-right
  • paddinng-bottom
  • padding-left

Todas as propriedades e preenchimentos podem ter os seguintes valores:

  • comprimento - especifica um preenchimento em px, pt, cm, etc.
  • % - especificaum preenchimento em % da largura do elemento que contém
  • inherit - especifica que o preenchimento deve ser herdado do elemento pai

Nota: Valores negativos não são permitidos.

Usando propriedades de preenchimento Individuais

Preeenchimento - propriedade abreviada

Para encurtar o código, é possível especificar todas as propriedades de preenchimneto em uma propriedade.

A padding propriedade é uma propriedade abreviada para as seguintes propriedades de preenchimento individuais:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Então, aqui está como funciona:
Se padding propriedade tiver quatro valores:

  • preenchimneto: 25px 50px 75px 100px
  • o preenchimento superior é 25px
  • o preenchimneto direito é 50px
  • o preenchimento inferior é 75px
  • o preenchimento esquerdo é 100px

A propriedade abreviada de preenchimento - 4 valores

Este elemento div tem um preenchimento superior 25px, um preenchimneto direito de 50px, um preenchimento inferior de 75px e um preenchimento esquerdo de 100px

Se a padding propriede tiver três valores:

  • preenchimento: 25px 50px 75px;
  • o preenchimento superior é 25px
  • os preenchimentos direito e esquerdo são 50px
  • o preenchimneto inferior é 75px

A propriedade abreviaa de preenchimneto - 3 valores

Este elemento div tem um preenchimeneto superior de 25px, um preenchimento direito e esquerdo de 50px e um preenchimneto inferior de 75px.

Se a padding propriedade tiver dois valores:

  • preenchimento superior e inferior são;
  • os Preeenchimento supeiror e inferior são 25px
  • os preeenchimento direito e esquerdo são 50px
Este elemento div tem um preenchimeneto supeiror e inferior de 25px e um preenchimneto direito e esquerdo de 50px

Se a padding propriedade tiver um valor:

  • preenchimeneto: 25px
  • todos os quatros preenchimentos têm 25px

A propriedae abreviada de preenchimento - 1 valor

Preenchimento e largura do elemento

A proriedadee CSS width especifica largura da área de conteúdo do elemento. A área de conteúdo é a parte dentro do preenchimneto, borda e margem de um elemento.

Portanto, se um elemento tiver uma largura especificada, o preenchimento adicionado a esse elemento será adicionado à largura total de elemento. Muitas vezes este é um resultado indesejável

Para manter a largura em 300px, não importa a quantidade de preenchimneto, você pode usar a box-sizing propriedade. Isto faz com que o elemento mantenha sua largura real; se você aumentar o preenchimento, o espaço de conteúdo disponével diminuirá
