Layout CSS - A propriedade display

A display propriedade é a propriedade CSS importante para controlar o layout

A propriedade de exibição

A display propriedade é usada para especificar como um elemento é mostrado em página da web.

Cada elemento HTML possui um valor de exibição padrão, dependendo do tipo de elemento. O valor de exibição padrão para a maioria dos elementos é block ou inline.

A display propriedade é usada para alterar o comportamento de exibição padrão dos elementos HTML


Elementos em nível de bloco

Um elemento de nível de bloco SEMPRE começa em uma nova linha e ocupa toda a largura disponível (estende-se para a esquerda e para a direita o máximo que puder).

O elemento div é um elemento de nível de bloco.

Exemplos de elementos em nível de bloco:

  • div
  • h1 - h6
  • formulário
  • cabeçalho
  • rodapé
  • rodapé
  • p

Elementos embutidos

Um elementos embutido não começa em uma nova linha e ocupa epena a largur necessária.

Este é um elemento span embutido dentro de um parágrafo.
Exemplo de elementos embutidos:

  • span
  • a
  • img

Os valores de propriedade de exibição

A display propriedade tem muitos valores:

Valores Descrição
inline: Exibe um elemento como um elemento inline(em linha).
block: Exibe um elemento como um elemento de bloco, como quebras de linha aontes e depois.
contents: Faz o contêiner desaparecer, tornando os elementos filhos filhos do elemento no níveç superior do DOM
flex: Exibe um elemento como um contêiner flexível de nível de bloco
grid: Exibe um elemento como um contêiner de grade de nível de bloco.
inline-block: Exibe um elemento como um contêiner de bloco em linha. O elemento em si é formatado como um elemento inline, mas você pode aplicar valores de altura e largura.
inline-grid: Exibe um elemento como um contêiner de grade de nível inline.
inline-table: Exibe o elemento como uma tabela de nível inline.
list-item: Faz com que o elemento se comporte como um elemento li.
run-in: Exibe um elemento como block ou inline, dependendo do contexto
table: Faz com que o elemento se comporte como um elemento table.
table-caption: Faz com que o elemento se comporte como um elemento caption
table-column-group: Faz com que o elemento se comporte como um elemento colgrup
table-header-group: Faz com que o elemento se comporte como um elemento thead
table-footer-group: Faz com que o elemento se comporte como um elemento tfoot
table-row-group: Faz com que o elemento se comporte como um elemento tbody
table-cell: Faz com que o elemento se comporte como elemento td
table-column: Faz com que o elemento se comporte como um elemento col
table-row: Faz com que o elemento se comporte como um elemento tr
none: O elemento é completamente removido (não exibido).
initial: Defina esta propriedade para seu valor padão
inherit: Herda esta propriedade do elemento pai.
