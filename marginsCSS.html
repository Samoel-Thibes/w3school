Margens CSS

As margens são usadas para criar espaços ao redor dos elementos, fora de quiasquer bordas definidas.

Este Elemento tem uma margem de 70px

CSS possui proriedades para especcificar margem de cada lado de um elemento:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Todas as proriedades de margem podem ter os seguitnes valores:

  • auto -o navegador calcula a margem
  • comprimento - especifica uma margem em px,pt,cm, etc.
  • % -especifica uma margem em% da largura do elemento que contém
  • inherit especifica que a margem deve ser herdada do elemento pai

A margin propriedade é uma pripriedade abreviada para as seguintes proriedades de margem individuais:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Então, aqui está como funciona:
Se a margin proprieadde tiver quatros valores:

  • Margem: 25px 50px 75px 100px
  • margem superior é 25px
  • margem direita é 50px
  • margem inferior é 75px
  • margem esquerda é 100px

Se a margin proprieddae tiver três valores:

  • Margem: 25px 50px 75px;
