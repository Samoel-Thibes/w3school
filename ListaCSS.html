Lista HTML e propriedades de lista CSS

Em HTML, existem dois tipos principais de lista:

  • listas não ordenadas (ul)- os itens de lsita são marcados com marcadores
  • lista ordenadas (ol)- os itens de lista são marcados com números ou letras

As propriedades da lista CSS permitem:

  • Defina diferentes marcadores de itens da lista para lista ordenadas
  • Defina diferentes marcadores de itens de lista para lista não ordenadas
  • Defia uma imagem como marcadore de item da lista
  • Adicione cores de fundo a lista e itens de lista

Diferentes marcadores de itens de lista

A list-style-type propriedade especifica o tipe de marcador de item de lista.

Posicione os marcadores de item da lista

A list-style-position propriedade especifica a posição dos marcadores de item da lsita (marcadores).


"posição no estilo de lista: fora;" significa que os marcadores estarão fora do item da lista. O início de cada linha de um item da lista será alinhado verticalmente. Este é o padro:

Lista - Propriedade abreviada

A list-sttyle propriedade é uma propriedade abreviada. É usado para definir todas as propriedades da lista em uma declaração:

Ao usar a propriedade, a ordem dos valores da propriedade é:

  • list-style-type (se uma imagem de estilo de lista for especifica, o valor desta propriedade será exibido se a imagem por algum motivo não puder ser exibida)
  • list-style-position (especifica se os marcadores de item de lista devem aparecer dentroi ou fora do fluxo de conteúdo)
  • list-style-imgage (especifica uma imagem como marcador de item da lista)

Se um dos valores de propriedada acima estiver faltando, o valor padrão da propriedade faltante será inserido, se houver.
